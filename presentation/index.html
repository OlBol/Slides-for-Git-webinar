Установка

  • https://git-scm.com/ – установка с официального сайта (для всех ОС)
  • apt-get install git
  • brew install git

На чем работать?

  • MacOS: открыть terminal
  • Linux: открыть terminal
  • Windows: открыть git bash

Проверка версии

  • git --version

Обновление

  • brew update && brew upgrade
  • apt-get update

Конфиг

  • Проверить свои настройки: git config --list
  • Указать свое имя:
    git config --global user.name "Olga Bolotova"
  • Указать свою жлектронную почту:
    git config --global user.email olbolot@gmail.com
